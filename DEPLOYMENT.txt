# Verity Labs - Vercel Deployment Guide

## Quick Deploy to Vercel

### Option 1: Deploy via Vercel CLI

1. Install Vercel CLI (if not installed):
   npm install -g vercel

2. Login to Vercel:
   vercel login

3. Deploy to production:
   vercel --prod

### Option 2: Deploy via GitHub + Vercel Dashboard

1. Push your code to GitHub (already done!)
2. Go to https://vercel.com/
3. Click "Add New Project"
4. Import your GitHub repository: veritylabsapp/veritylabs
5. Vercel will auto-detect settings
6. Click "Deploy"

### Option 3: Deploy via Vercel Dashboard (Direct Upload)

1. Go to https://vercel.com/new
2. Click "Browse" and select your project folder
3. Click "Deploy"

## Files Ready for Deployment

✅ index.html - Main website file (production-ready)

✅ package.json - Project metadata
✅ .gitignore - Git ignore rules

## Custom Domain Setup

After deployment:
1. Go to your project in Vercel Dashboard
2. Click "Settings" > "Domains"
3. Add your custom domain: veritylabs.com
4. Follow Vercel's DNS configuration instructions
